部署手册
====================

github 项目 `kubernetes-the-hard-way  <https://github.com/kelseyhightower/kubernetes-the-hard-way>`_ 详细介绍了如何一步步安装 kubernetes，但如果你想快速的部署 kubernetes 集群环境并且已经对 ansible 非常的熟悉，可以使用 github 项目 `kubespray <https://github.com/kubernetes-incubator/kubespray>`_ 来部署环境。在此手动部署 kubernetes 细节不在赘述。
